Generate google maps before or after post,popup or using shortcode.

== Description ==

j2Gomap allows you to add interactive google map to your post or page using address not coordinates(longhitude and latitude),you can display the google before or after in your single post, popup or using shortcode.


== Installation ==

1. Upload the `j2gomap` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure your j2gomap settings under Settings > j2gomap

That's it! Now you can easily start add google map in your Wordpress site.
